# Space Tourism React + Tailwind + TypeScript

Frontend Mentor Space Tourism website built with React, Tailwind CSS, React Router, and TypeScript.

## Run the project

```bash
npm install
npm run dev
```

## Folder structure

```txt
src/
  components/
  data/
  pages/
  types/
    spaceTourism.ts
```

The `types` folder contains the interfaces for the data coming from `data.json`.
